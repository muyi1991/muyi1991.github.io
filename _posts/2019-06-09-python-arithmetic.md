#### 加、减、乘、除、取余
整数除的出的数是浮点型

```
>>> 1+9
10
>>> 7-4
3
>>> 4*3
12
>>> 8/2
4.0
>>> 8%2
0
```

#### 取幂运算

```
>>> 2**3
8
>>> 4**2
16
```

#### 向下取整最接近的整数


```
>>> 9//2
4
```

#### 求平均数

```
>>> print((30+31+32+33+34)/5)
32.0
```
