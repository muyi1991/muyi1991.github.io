#### 布尔运算符
不等于是“!=”

```
>>> 1>2
False
>>> 1<2
True
>>> 1>=2
False
>>> 1<=2
True
>>> 1==2
False
>>> 1!=2
True
```

#### 逻辑运算法
and 全部满足
or 至少有一个满足
not 布尔值反转

```
>>> 1<2 and 1>2
False
>>> 1<2 and 2<3
True
>>> 1<2 or 4>5
True
>>> 1>2
False
>>> not 1>2
True
```
